# 文献管理系统 — 功能介绍与操作手册

本地化科研文献管理工具，采用标签分类 + PDF 批注 + 附件管理的模式，替代 Zotero 的繁琐目录结构，同一文献可归属多个标签。

启动后浏览器访问 `http://127.0.0.1:8088`。右上角 ⚙️ 齿轮按钮可切换日/夜间模式和中英文界面。

## 环境要求

- Python 3.10+
- Windows / macOS / Linux

## 安装与启动

点击右上角 ⚙️ 齿轮按钮可切换日/夜间模式和中英文界面。

### 方式一：Python 源码运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动
python main.py
```

### 方式二：打包为独立 exe（推荐给非技术用户）

```bash
# 1. 在本机打包
pip install pyinstaller
build.bat           # 或双击 build.bat

# 2. 把 dist\文献管理系统\ 整个文件夹拷给其他人
# 3. 对方双击 文献管理系统.exe 即可运行
```

打包后对方**不需要安装 Python**，所有依赖都打包在 `_internal` 目录中。
数据（数据库、PDF、附件）保存在 exe 同级目录下，迁移时复制整个文件夹即可。

浏览器将自动打开 `http://127.0.0.1:8088`。如果未自动打开，手动访问该地址。

## 目录结构

```
literature-manager/
├── main.py              # 入口，启动服务器
├── config.py            # 路径配置
├── database.py          # 数据库连接
├── models.py            # 数据模型（文献/标签/批注/附件）
├── schemas.py           # 请求响应结构
├── routers/             # API 接口
├── services/            # Zotero 导入器 & 注释烘焙
├── static/              # 前端（HTML/CSS/JS）
├── data/literature.db   # SQLite 数据库（自动创建）
├── papers/              # 文献 PDF 存储
├── attachments/         # 附件存储（按文献 ID 分目录）
└── requirements.txt
```

## 二、文献管理

### 添加文献

点击「+ 添加文献」→ 选择 PDF → 系统自动提取标题（优先 PDF 元数据，备选文件名）→ 可选手动编辑 → 点「添加」。

添加时自动检测库内是否已有相同标题的文献，如发现重复会给出警告。

> PDF 文件统一存放在 `papers/` 目录中，以原文件名保存。

### 文献卡片

选中文献后上方卡片高亮，下方预览面板展示该文献的批注、元数据、笔记。

- **单击**卡片 → 选中并显示预览
- **双击**卡片 → 打开 PDF 阅读器
- 卡片第一页缩略图会自动生成（DPI 120）

### 快速编辑

预览面板直接编辑：

| 字段 | 方式 |
|------|------|
| 标题 | 点击标题文字直接修改，回车或失焦即保存 |
| 期刊、作者、年份、DOI | 点击输入框修改，失焦即保存 |
| 笔记 | 自由输入，停止输入 0.6 秒后自动保存 |
| 标签 | 下拉选择添加，点击标签 × 移除 |
| 删除文献 | 标题栏右侧红色按钮 |

### 删除文献

预览面板中点击「删除」按钮，确认后同时删除数据库记录和 `papers/` 中的 PDF 文件。

> 如果手动从 `papers/` 中删除了 PDF，刷新页面后对应卡片自动消失，数据库中记录也会被清理。

## 三、标签分类

### 标签树

左侧树形标签支持两级层级。顶部常驻「未分类」虚拟标签，显示所有未打标签的文献。

- **新建**：点击标签树上方的 `+` 按钮
- **编辑**：悬停标签 → 点击铅笔图标 ✎
- **删除**：悬停标签 → 点击垃圾桶图标 🗑

### 颜色预设

编辑标签时颜色输入框下方提供 4×4 色块预设，点击即选。

### 文献关联

- 预览面板中通过「+ 标签」下拉框为文献添加标签
- 悬停标签徽章点击 × 移除关联
- 同一文献可关联多个标签

## 四、设置（⚙️ 齿轮按钮）

右上角 ⚙️ 齿轮按钮点击弹出设置菜单：

- **日夜切换**：点击切换亮色/暗色模式

齿轮按钮可拖拽移动到屏幕任意位置，位置自动保存。

## 五、PDF 阅读器

双击文献卡片或点击预览面板批注即可打开 PDF 全屏阅读器。

### 工具栏

| 按钮 | 功能 |
|------|------|
| ← 返回列表 | 退出阅读器 |
| 高亮 | 选中文字后自动创建黄色高亮批注 |
| 下划线 | 选中文字后创建下划线批注 |
| 方框 | 拖拽鼠标画出矩形框标注 |
| 批注 | 点击页面任意位置弹出输入框，创建便签 |
| 指针 | 正常选择/复制文字模式 |
| 🎨 | 颜色选择器，影响接下来的批注颜色 |
| − / 150% / + | 缩放（50% – 400%） |
| ◀ / 页码 / ▶ | 页面导航 |
| 📎 附件 | 上传 .txt / .doc / .docx 附件到该文献 |

### 批注系统

- 所有批注存储在数据库中，**不修改原始 PDF 文件本身**
- 创建后自动保存，刷新不丢失
- 右侧批注列表点击条目 → 跳转到对应页面
- 点击批注弹出气泡 → 可查看内容和删除
- 悬停批注条目 → 出现删除按钮

### 文字层

PDF 阅读器包含**文字选择层**，可直接选中 PDF 文本并复制（需要 PDF 本身含文本层，扫描件除外）。

### 右侧面板（三区）

**笔记**：自由编辑，停止输入 0.5 秒自动保存，与主界面预览面板笔记实时同步。点击「导出」按钮下载为 TXT。

**批注列表**：按页码分组展示该文献所有批注。

**附件列表**：展示该文献所有上传的附件，点击可预览文本内容或下载。

## 六、附件管理

### 在主界面

预览面板中的「附件」区可直接上传和下载附件。

### 在 PDF 阅读器中

点击 📎 附件按钮上传，附件列表展示在右侧面板底部。点击附件名弹出内容预览（支持 txt / doc / docx / md / csv / json 等文本格式），非文本格式提示下载。

## 七、从 Zotero 迁移

点击左下角「从 Zotero 导入」：

1. 系统**自动检测**本机 Zotero 数据目录（`C:\Users\用户名\Zotero`）
2. 若自动检测失败，手动输入包含 `zotero.sqlite` 和 `storage` 的目录路径
3. 点击「扫描预览」列出可导入文献及注释数量
4. 预览中标黄标记已存在于库中的文献（将自动跳过）
5. 确认导入

导入过程中：
- PDF 从 Zotero storage 复制到 `papers/`
- Zotero 注释（高亮、下划线、删除线、文本批注）**永久合并写入 PDF 文件**
- 元数据（标题、作者、年份、期刊、DOI）自动提取
- 已存在的文献自动跳过

## 十、常见问题

**Q: PDF 阅读器中文字无法选中？**
A: 确认工具栏选择的是「指针」模式。扫描件 PDF（纯图片）不含文本层，选中功能不适用。

**Q: 导入 Zotero 后批注没显示？**
A: 导入时会写入 PDF 内部，通过 PyMuPDF 可读取。请在 PDF 阅读器中查看，或检查源 Zotero 是否有批注。

**Q: papers/ 中的 PDF 被我删了怎么办？**
A: 刷新页面，系统自动清理无效记录，只显示实际存在的 PDF。

**Q: 如何迁移到另一台电脑？**
A: 复制整个 `literature-manager` 文件夹即可，`data/`、`papers/`、`attachments/` 包含所有数据。
